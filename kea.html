Theme 04: Basic UX

04.04.01. Design sprint: Native app, 04.04.02. Theme documentation

In this theme we needed to combine all our knowledge that we gained in the last couple of weeks and in a group make a new improvement for KEA by designing a new app.

By using design sprint technique we were able to find a great solution because we all trusted in it's effectiveness. Even though the start was rough due to many ideas, by using voting every time we got through it.

The documentation holds all work from the start of theme. I understood how important is to documment every single detail so the process could work more smoothly and it would be easy to look back if something went wrong along the way and improve.
