Theme 03: Basic Animation

03.04.05. Link to final interactive production, 03.04.06. Link to screen cast documentation

In this theme we got a group project to make an interactive animation game based on a scene from a movie, novel or story. Our group was assigned Cinema Noir style and a song "Ain't No Cure For Love" by Leonard Cohen. My role was a designer, so I illustrated most of the graphics which led me to improve my skills and try different styles of drawing so it would fit Cinema Noir

This project was the most precise one yet at that moment because we needed to combine all our previous knowledge and skills such as creating the story, storyboard, making sequence diagram, all the illustrations and coding the whole thing.

During the project we did the peer review to test our animation usability and later on made some changes to improve interaction with the user due to experts not fully understanding what was interactive. This test also let me personally improve the illustrations for them to fit the mysterious scene more.
