Theme 03: Basic Animation

03.01.02. Link to simple animation

In this theme I learned about JavaScript and how I can implement it to a website. We were introduced to how simple animations are created using spritesheets and that movement of objects are not only based on CSS with keyframes, transitions and transform.

Learning about variables was the next step in my coding knowledge. Creating elements, functions which trigger keyframes made creating animation much more interesting. Using console to see errors helped to fix the mistakes.

Learning about classList and EventListener made me realise how many websites that I use daily work, when one mouse click can activate functions.
